# 🇱🇦 벨로루트 비엔티안 (VeloRoute Vientiane)
> **라오스 비엔티안 우기 침수 대응 실시간 홍수 회피 및 오토바이/뚝뚝 특화 스마트 내비게이션**

본 프로젝트는 라오스 비엔티안 중심가와 메콩강 주변의 고유한 기후 및 교통 인프라 문제를 스마트시티 관점에서 해결하기 위해 설계된 실시간 안전 길찾기 서비스 프로토타입입니다.

특히, 기술적 구현을 넘어 공공 재난 매뉴얼의 효과적인 전달을 위해 **국어 교육과정의 '설득 전략' 및 '표현 전략'을 UI/UX 및 기획에 연동 설계**하고, 이를 **실시간 한국어 ↔ 라오어(Lao) 양방향 번역 시스템**으로 구현하여 해커톤적 혁신성과 인문학적 설득력, 글로벌 사용성을 동시에 확보하였습니다.

---

## 💡 국어과 연계 설득 및 표현 전략 설계 (Rhetoric & Communication Design)
재난 정보는 기술적으로 정확하게 계산하는 것만큼이나, **사용자가 신뢰하고(인성), 논리적으로 납득하여(이성), 행동을 바꾸도록 자극(감성)**하는 설득 전략이 중요합니다. 본 프로젝트는 UI와 알림 메시지에 이를 적극 투입하고, 실시간 토글을 통해 한국어와 라오어 모두에 동일한 언어적 가치가 구현되도록 매핑하였습니다.

```
                  [ 설득 및 표현 전략 매핑 구조 ]
                     
         ┌─────────────────── 설득 전략 (Persuasion) ───────────────────┐
         │                                                             │
   [인성적 설득: Ethos]             [이성적 설득: Logos]            [감성적 설득: Pathos]
   - NDMC/DMH 국가인증 마크        - 수문학적 안전성 지수 제공     - 위기 상황 속 생명 존중 메시지
   - 정보 제공 시스템 권위 확보     - 객관적 수치 및 우회 데이터     - 사이렌 점멸 시각 시각화
         │                                                             │
         └─────────────────── 표현 전략 (Rhetorical) ───────────────────┘
                     ┌───────────────────┴───────────────────┐
             [관습적 표현: Idiom]                    [창의적 표현: Creative]
             - 친숙한 속담 인용 및 변형               - 대조와 비유를 통한 경각심 극대화
             - "돌다리도 두드려보고...", "소 잃고.."  - "진흙탕 지름길은 시간을 지르지..."
```

### 1. 설득 전략 (아리스토텔레스의 수사학 3요소)
* **인성적 설득 전략 (Ethos - 신뢰성과 권위)**
  * **기획 설계**: 사용자에게 실시간 경로를 안내할 때 신뢰를 주지 못하면 안내에 따르지 않습니다.
  * **한국어 적용**: `"🛡️ 라오스 기상 DMH / 국가재난 NDMC 인증 안전 규격 준수"`
  * **라오어 적용**: `"ປະຕິບັດຕາມມາດຕະຖານຄວາມປອດໄພທີ່ໄດ້ຮັບການຮັບຮອງຈາກ DMH ແລະ NDMC ລາວ"`
  * **UI 반영**: 경로 피드 및 정보 섹션에 인증 라벨을 탑재하여 시스템의 공신력을 획득했습니다.
* **이성적 설득 전략 (Logos - 논리와 수치)**
  * **기획 설계**: 왜 이 길로 돌아가야 하는지를 이성적이고 귀납적인 통계 데이터로 입증합니다.
  * **한국어 적용**: `"경로 안전성 지수: 99.8% (최적)"` / `"최대 침수 조우 깊이: 0.120 m"`
  * **라오어 적용**: `"ດັດຊະນີຄວາມປອດໄພເສັ້ນທາງ: 99.8% (ດີທີ່ສຸດ)"` / `"ຄວາມເລິກນ້ຳຖ້ວມສູງສຸດ: 0.120 ແມັດ"`
  * **UI 반영**: 예상 경로 피드에 통계 분석을 논리적으로 제시하여 우회의 당위성을 증명합니다.
* **감성적 설득 전략 (Pathos - 감정과 연대감 자극)**
  * **기획 설계**: 긴급 대피 단계에서 사용자에게 강력한 행동적 동기(대피 행동)를 일으키기 위해 생명 존중과 가족애를 자극합니다.
  * **한국어 적용**: `"세상의 어떤 보물도 당신의 소중한 생명만큼 값지지 않습니다. 사이렌이 울리는 지금, 망설이지 말고 인근 고지대 대피소로 즉시 대피하여 안전을 지키십시오. 당신의 무사 귀환이 가족에겐 가장 큰 기쁨입니다."`
  * **라오어 적용**: `"ບໍ່ມີຊັບສົມບັດໃດໆໃນໂລກນີ້ທີ່ຈະມີຄ່າຫຼາຍໄປກວ່າຊີວິດທີ່ຮັກແພງຂອງທ່ານ. ໃນຂະນະທີ່ສຽງໄຊເລນດັງຂຶ້ນນີ້, ຢ່າລັ່ງເລໃຈ, ຈົ່ງຮີບອົບພະຍົບໄປຫາສູນອົບພະຍົບທີ່ຢູ່ເຂດເນີນສູງທັນທີເພື່ອຄວາມປອດໄພຂອງທ່ານ. ການກັບມາຢ່າງປອດໄພຂອງທ່ານແມ່ນຄວາມສຸກທີ່ຍິ່ງໃຫຍ່ທີ່ສຸດຂອງຄອບຄົວ."`
  * **UI 반영**: 최상단 붉은 사이렌 배너 경보 발령 시 표출되어 공포를 해소하고 따뜻한 연대감으로 대피를 독려합니다.

### 2. 표현 전략 (표현의 적절성과 참신성)
* **관습적 표현 전략 (격언과 속담의 유연한 활용)**
  * **기획 설계**: 사용자에게 가장 익숙하고 상식적인 속담을 활용하여 안전 수칙에 대한 수용도를 높입니다.
  * **한국어 적용**: 맑은 날(NORMAL) 배너: **"돌다리도 두드려보고 건너라"** 했습니다. / 호우 시(WARNING) 배너: **"소 잃고 외양간 고친다"** 하였습니다.
  * **라오어 적용**: 맑은 날(NORMAL) 배너: **"ຂ້າມຂົວຫີນກໍຕ້ອງເຄາະເບິ່ງກ່ອນ"** (돌다리도 두드려보고 건너야 한다) / 호우 시(WARNING) 배너: **"ເສຍງົວແລ້ວຈຶ່ງເຮັດຄອກ"** (소 잃고 외양간 고친다)
  * **UI 반영**: 각 상황에 매칭되는 한국 및 라오스 현지 속담을 기상 재난 배너에 표출하여 직관성과 친근감을 증대시켰습니다.
* **창의적 표현 전략 (참신한 은유와 대조법)**
  * **기획 설계**: 상투적인 경고에서 벗어나 문학적 대조와 비유를 사용해 경각심을 극대화합니다.
  * **한국어 적용**: 주의 단계(CAUTION) 배너: **"진흙탕 지름길은 당신의 시간을 지르지 못하고 오히려 삼켜 버립니다."**
  * **라오어 적용**: 주의 단계(CAUTION) 배너: **“ທາງລັດທີ່ເປັນຕົມບໍ່ໄດ້ຊ່ວຍປະຢັດເວລາຂອງທ່ານ, ແຕ່ມັນຈະກືນກິນເວລາຂອງທ່ານ.”**
  * **UI 반영**: 시간 단축을 위해 무리하게 강변 비포장도로(야시장 골목 등)로 진입하려는 운전자들의 행동을 효과적으로 억제합니다.

## 🔄 실시간 한국어-라오어 번역 토글 시스템 (Real-time Bilingual Switch)
해커톤의 글로벌 가치와 비엔티안 현지 적용성을 완벽히 구현하기 위해, **전체 웹페이지 요소에 대한 실시간 번역 스위치**를 구현했습니다.

* **원클릭 언어 스위치**: 브랜드 헤더 최상단에 HSL 슬레이트 테마의 국기 토글 버튼(`🇰🇷 KO / 🇱🇦 LO`)을 배치하여 언제든 양방향 언어 전환이 가능합니다.
* **전체 컴포넌트 동기화**: 번역 헬퍼 `t()`와 노드 번역 헬퍼 `getNodeLabel()`을 사용하여 헤더, 사이드바, 컨트롤창, 도로 상태, 재난 정보, 지도 팝업, 제보 모달까지 100% 번역됩니다.
* **백엔드 리스폰스 동적 번역**: 백엔드 파이썬 서버로부터 반환되는 경로 안내 코멘트(Remarks) 및 에러 메시지(예: `주의: 최대 0.15m 깊이의 침수...`, `안전한 대피로가 존재하지 않습니다...`)까지 프론트엔드의 파서 및 번역 헬퍼가 실시간 감지하여 자연스러운 라오어로 매끄럽게 번역 출력합니다.

## 🌟 주요 기능 (Key Features)

1. **실시간 기상 재난 경보 배너**: 강수량/수위에 맞춘 가변 텍스트 경보 및 설득/표현 전략 적용 (실시간 다국어 지원).
2. **긴급 대피소 최적 경로 안내**: 녹색 방패 마커 대피소 4곳(`I`, `Q`, `V`, `P`) 및 최인접 안전 대피로 안내.
3. **실시간 사용자 장애물 제보 및 우회**: 사고, 침수, 포트홀, 통제 구역을 클릭 제보하여 경로 탐색 시 가중치 패널티(20배) 부여로 우회 경로 자동 연산.
4. **차량 종류별(뚝뚝, 오토바이, 승용차) 특화 경로 계산**: 비포장 패널티 가중치(뚝뚝 6.5배, 오토바이 2.5배) 및 침수 한계점(뚝뚝 15cm, 오토바이 22cm) 설정.

## 🛠️ 기술 스택 (Tech Stack)

### Backend
* **Python 3.13** & **FastAPI**
* **Dynamic Dijkstra Routing Algorithm** (사용자 제보 및 침수 깊이 실시간 반영 가중치 그래프)
* **Hydrological Telemetry Simulator** (강수량 및 메콩강 범람 연동 도로 침수 수위 수치 시뮬레이터)

### Frontend
* **React** (Vite 번들러)
* **React-Leaflet** (Futuristic Dark Matter 테마 지도 연동)
* **Lucide React** (프리미엄 반응형 아이콘셋)

### Infrastructure (OSRM)
* **OSRM (Open Source Routing Machine)**: 뚝뚝에 특화된 비포장 Tag 회피용 LUA 프로파일(`tuktuk.lua`) 설계 및 라오스 OSM 지리 데이터 정적 가중치 전처리 파이프라인 수립.

## 🚀 실행 및 테스트 방법 (How to Run)

### 1. 백엔드 (FastAPI) 구동
```bash
cd backend
python -m venv venv
source venv/Scripts/activate  # Windows 환경
pip install -r requirements.txt
python -m uvicorn main:app --host 0.0.0.0 --port 8001 --reload
```
* 백엔드는 포트 **8001**에서 API 서비스를 개시합니다.
* API 문서(Swagger UI)는 [http://localhost:8001/docs](http://localhost:8001/docs)에서 직접 제어하실 수 있습니다.

### 2. 프론트엔드 (React) 구동
```bash
cd frontend
npm install
npm run dev
```
* 프론트엔드는 포트 **3000**에서 구동됩니다.
* 브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 실시간 맵을 조작할 수 있습니다.

### 3. 알고리즘 테스트 실행
```bash
python backend/test_router.py
```
* 건조기 비포장 기피, 우기 강변 침수 우회, 극단적 폭우 시 고립, 실시간 장애물(사고) 제보 시 우회 등 4대 경로 체크 시나리오에 대해 알고리즘 무결성을 검증합니다.
