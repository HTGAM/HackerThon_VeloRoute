# 🇱🇦 벨로루트 비엔티안 (VeloRoute Vientiane)
> **라오스 비엔티안 우기 침수 대응 실시간 홍수 회피 및 오토바이/뚝뚝 특화 스마트 내비게이션**

본 프로젝트는 라오스 비엔티안 중심가와 메콩강 주변의 고유한 기후 및 교통 인프라 문제를 스마트시티 관점에서 해결하기 위해 설계된 실시간 안전 길찾기 서비스 프로토타입입니다.

**실시간 한국어 ↔ 라오어(Lao) 양방향 번역 시스템**을 탑재하여 현지 주민과 관광객의 사용성과 접근성을 극대화하고, 해커톤적 혁신성과 비엔티안 현지 실질적 공공 가치를 확보하였습니다.

---

## 🔄 실시간 한국어-라오어 번역 토글 시스템 (Real-time Bilingual Switch)
해커톤의 글로벌 가치와 비엔티안 현지 적용성을 완벽히 구현하기 위해, **전체 웹페이지 요소에 대한 실시간 번역 스위치**를 구현했습니다.

* **원클릭 언어 스위치**: 브랜드 헤더 최상단에 HSL 슬레이트 테마의 국기 토글 버튼(`🇰🇷 KO / 🇱🇦 LO`)을 배치하여 언제든 양방향 언어 전환이 가능합니다.
* **전체 컴포넌트 동기화**: 번역 헬퍼 `t()`와 노드 번역 헬퍼 `getNodeLabel()`을 사용하여 헤더, 사이드바, 컨트롤창, 도로 상태, 재난 정보, 지도 팝업, 제보 모달까지 100% 번역됩니다.
* **백엔드 리스폰스 동적 번역**: 백엔드 파이썬 서버로부터 반환되는 경로 안내 코멘트(Remarks) 및 에러 메시지(예: `주의: 최대 0.15m 깊이의 침수...`, `안전한 대피로가 존재하지 않습니다...`)까지 프론트엔드의 파서 및 번역 헬퍼가 실시간 감지하여 자연스러운 라오어로 매끄럽게 번역 출력합니다.

## 🌟 주요 기능 (Key Features)

1. **실시간 기상 재난 경보 배너**: 강수량/수위에 맞춘 가변 텍스트 경보 및 설득/표현 전략 적용 (실시간 다국어 지원).
2. **긴급 대피소 최적 경로 안내**: 녹색 방패 마커 대피소 4곳(`I`, `Q`, `V`, `P`) 및 최인접 안전 대피로 안내.
3. **실시간 사용자 장애물 제보 및 우회**: 사고, 침수, 포트홀, 통제 구역을 클릭 제보하여 경로 탐색 시 가중치 패널티(20배) 부여로 우회 경로 자동 연산.
4. **차량 종류별(뚝뚝, 오토바이, 승용차) 특화 경로 계산**: 비포장 패널티 가중치(뚝뚝 6.5배, 오토바이 2.5배) 및 침수 한계점(뚝뚝 15cm, 오토바이 22cm) 설정.
5. **실시간 대피 지연 시간 골든타임 알림**: 실시간 수위 상승 속도($v_{\text{rise}}$)와 차종별 임계 침수 고도를 결합해 대피 가능 잔여 골든타임을 동적으로 산출 경고.
6. **AI 물리학적 대피 가이드**: 아르키메데스의 부력 공식, 유체 저항, 타이어 접지 마찰 변화율 등의 수문 물리 엔진 데이터 해석 및 가이드 제공.
7. **실시간 대피 시뮬레이션 미니 게임 (VeloRoute Flood Escape Challenge)**: 폭우 속에서 차량(뚝뚝 등)을 WASD/방향키 또는 터치패드로 조작하여 대피소로 대피하는 챌린지 모드 (리더보드 및 사운드 효과 완비).
8. **실시간 내비 피드 HUD 및 인포그래픽 게이지**: 안전성 지수 바, 한계 대비 조우 깊이 게이지 바, 물리 역학 지표 2x2 그리드 및 실시간 접지 마찰력 게이지 시각화.
9. **전역 줄바꿈 규칙 최적화 및 3색 LED 인디케이터**: `word-break: keep-all` 전역 적용으로 텍스트의 어절 잘림 현상을 완벽 해결하고, 실시간 도로 상태 피드에 3색 LED 표시등을 적용해 직관적 가시성을 극대화.

## 🛠️ 기술 스택 (Tech Stack)

### 🔤 사용 언어 (Languages)
* **Python**: 백엔드 API 서버 설계, 수역학 시뮬레이션 물리 모델 구축, 다익스트라 경로 안내 핵심 알고리즘 개발.
* **JavaScript (ES6+)**: React 기반의 반응형 대시보드 및 웹 GIS 사용자 인터페이스 구현.
* **Lua**: OSRM(Open Source Routing Machine) 기판 지리 정보 분석을 위해, 뚝뚝 및 오토바이 특성에 맞춤 설계된 커스텀 가중치 필터링 프로파일(`tuktuk.lua`) 스크립팅.
* **HTML5 / CSS3**: 전술 지휘통제소 스타일의 네온 테마 및 글래스모피즘(Glassmorphism) 시각 디자인 마크업.
* **Batch (Windows Batch Script)**: 복잡한 빌드/설치 단계를 한 번에 실행시키는 원클릭 무설정 오토매틱 런처(`run.bat`) 스크립팅.

### ⚙️ 백엔드 (Backend)
* **FastAPI**: 초고속 비동기 ASGI 웹 프레임워크 기반 REST API 서버.
* **Uvicorn**: 고성능 파이썬 ASGI 웹 서버.
* **Dynamic Dijkstra Algorithm**: 사용자 재난 제보(실시간 우회 페널티 20배) 및 실시간 가변 침수 깊이(수위별 통행 불가 차단 및 최대 9배 서행 연산)가 반영되는 동적 가중치 알고리즘.
* **Hydrological Telemetry Simulator**: 몬순 우기 강수량(mm/h)과 메콩강 수위 변화에 따른 도로별 실시간 침수 고도를 모의 연산하는 데이터 시뮬레이터 파이프라인.

### 💻 프론트엔드 (Frontend)
* **React**: 사용자 중심 컴포넌트 기반 UI 라이브러리.
* **Vite**: 초고속 프론트엔드 빌드 도구 및 개발 서버.
* **React-Leaflet / Leaflet**: 지도 렌더링, 오프라인 호환 노드/링크 폴리라인 및 사용자 제보 마커 맵핑 웹 GIS 프레임워크 (Futuristic Dark Matter 맵 스킨).
* **Lucide React**: 현대적인 시스템 아이콘 패키지.

### 🧠 인공지능 (AI Integration)
* **Google Gemini API (gemini-2.5-flash)**: 인앱 키 설정을 연동하여, 현재 사용자의 날씨 파라미터, 대피 대상 차량, Dijkstra로 탐색된 긴급 대피 경로(거리, 침수 비고) 등의 풍부한 실시간 물리 컨텍스트를 학습하고 재난 대처 가이드를 대화형으로 제공하는 지능형 비서.

### 🐳 컨테이너화 및 인프라 (Infrastructure & DevOps)
* **Docker & Docker Compose**: 멀티컨테이너 가상화 아키텍처 환경 구축.
  - **`osrm-backend`**: 라오스 전체 OpenStreetMap(OSM) 지리 원천 데이터를 컴파일하고 커스텀 Lua 프로파일을 구동하는 지리 라우팅 백본 엔진.
  - **`routing-api`**: OSRM 지리 정보와 실시간 침수 시뮬레이터를 병합해 최적 경로를 도출하는 FastAPI 백엔드 컨테이너.
  - **`frontend`**: React 클라이언트 정적 파일을 서비스하고 챗봇을 연결하는 Vite 프론트엔드 컨테이너.

## 🚀 실행 및 테스트 방법 (How to Run)

### 0. 원클릭 일괄 실행 (Windows 추천) [NEW]
프로젝트 루트 디렉토리에 위치한 배치 파일을 실행하면 백엔드 가상환경 설정, 의존성 설치, 프론트엔드 빌드 및 서버 구동, 브라우저 새 창 실행까지 **자동으로 한 번에 처리**됩니다.
```bash
# 루트 디렉토리에서 배치 파일 실행
.\run_veloroute.bat
```
* 해당 스크립트는 백엔드(포트 8001) 및 프론트엔드(포트 3000)를 병렬 구동하고 웹 브라우저를 자동 실행합니다.

### 1. 개별 수동 구동 - 백엔드 (FastAPI)
```bash
cd backend
python -m venv venv
source venv/Scripts/activate  # Windows 환경
pip install -r requirements.txt
python -m uvicorn main:app --host 0.0.0.0 --port 8001 --reload
```
* 백엔드는 포트 **8001**에서 API 서비스를 개시합니다.
* API 문서(Swagger UI)는 [http://localhost:8001/docs](http://localhost:8001/docs)에서 직접 제어하실 수 있습니다.

### 2. 개별 수동 구동 - 프론트엔드 (React)
```bash
cd frontend
npm install
npm run dev
```
* 프론트엔드는 포트 **3000**에서 구동됩니다.
* 브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 실시간 맵을 조작할 수 있습니다.

### 3. 알고리즘 테스트 실행
```bash
python backend/test_router.py
```
* 건조기 비포장 기피, 우기 강변 침수 우회, 극단적 폭우 시 고립, 실시간 장애물(사고) 제보 시 우회 등 4대 경로 체크 시나리오에 대해 알고리즘 무결성을 검증합니다.
